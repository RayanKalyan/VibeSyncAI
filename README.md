# VibeSync AI 🎵

**VibeSync AI** is a high-fidelity, AI-powered music recommendation engine developed by **Andesite Studios**. It bridges the gap between raw human emotion and digital sonic frequencies by leveraging generative AI to curate personalized Spotify playlists.

## 🚀 The Core Engine
VibeSync bypasses traditional keyword-based search by using **Gemini 1.5-Flash** to translate natural language prompts—like *"midnight drifting"* or *"melancholic synthwave"*—into precise audio feature vectors, which are then synthesized into real-time Spotify recommendations.

## 🛠 Tech Stack
* **Frontend:** React, Vite, Tailwind CSS
* **AI Pipeline:** Google Gemini API (Generative Content)
* **Authentication:** Spotify OAuth 2.0 with PKCE (Proof Key for Code Exchange)
* **Infrastructure:** Deployed via Vercel

## 💎 Features
* **Cinematic UI:** Built with glassmorphism, dynamic ambient lighting, and high-performance micro-interactions.
* **Global Market Override:** Bypasses regional content restrictions to fetch worldwide music trends.
* **Ghost Easter Eggs:** Includes hidden data vectors (2057/31) that populate based on session activity.
* **Privacy First:** Built with secure PKCE handling to ensure user credentials are never exposed in the URL.
